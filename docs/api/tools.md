# Built-in Tools

> Reference tool implementations live in [`examples/tools/`](https://github.com/iciouss/llm-framework-core/tree/main/examples/tools). They are not part of the library API — copy them into your project as a starting point or use them as runnable examples.

The library ships the `@tool` decorator and the `@cached_tool` variant; concrete tools are reference implementations, not a fixed inventory. See `examples/tools/` for filesystem, shell, web fetch, calculator, clock, and memory tools.
